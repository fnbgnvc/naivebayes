# naivebayes
naive bayes classifier in java.
To run in command line:
Navigate to the directory that contains the .jar file. Run using java -jar Bayes.jar [args]
The program takes two arguments, the first being the name of the training data file,(e.g. breast-cancer-training.csv)
 and the second being the name of the test file (e.g. breast-cancer-test.csv). 
 It outputs the predicted and actual class for each instance.
